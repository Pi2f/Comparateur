# Comparateur de bières 

Ce comparateur a pour but de ravir les amateurs de bières.
**Buts** :
Comparer les bières entre elles selon :
* Brune, blonde, blanche, ambrée (type)
* Bouteille ou canette (contenant)
* Degré d'alcool
* Pays
* Artisanal
* Prix
* Volume
* Avis des consommateurs



Grands concepts :
* Bière
* Total point
* Recherche
* Classement
